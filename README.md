# Storytelling Project – Creative Writing Lab (C2 English)

Storytelling Project is an ongoing personal initiative where I use creative writing
as a way to strengthen two core skills:

- Public presentations
- Entrepreneurial and leadership communication

Through short stories written mainly in English (C2 level), and occasionally
including French or Chinese references, I experiment with how narratives:

- Capture attention
- Simplify complex ideas
- Create emotional connection
- Inspire action

This repository is open to readers, learners, and professionals interested
in storytelling as a tool for communication, leadership, and innovation.
